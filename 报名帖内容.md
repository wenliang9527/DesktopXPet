## 项目简介

DesktopXPet 是一款 **AI 工作监控桌面宠物**，通过可爱的像素风角色实时展示你的 AI 编码工具工作状态。

让开发者在等待 AI 生成代码、处理任务时，不再枯燥等待——桌面上那只小猫咪/剑士/Q版女孩会根据 AI 的状态自动切换动画：忙碌时努力工作、完成时开心庆祝、出错时沮丧提醒、闲置时悠闲发呆甚至睡觉！

---

## 核心功能

### 1. 透明悬浮宠物窗口
- 无边框透明窗口，始终置顶
- 鼠标穿透（仅角色区域可交互）
- 支持拖拽移动，位置自动记忆

### 2. 实时状态监控
- 通过插件系统接入多种 AI 工具
- 内置插件：System（系统资源）、GitHub、Ollama
- 支持自定义扩展

### 3. 精灵图动画引擎
- Canvas 2D 渲染，支持高 DPI
- 像素风锐利放大
- 5 种状态动画：idle、working、happy、sleeping、error

### 4. 皮肤系统
- 内置 3 套皮肤：橘猫、蝴蝶剑士、Q版女孩
- 支持本地目录加载自定义皮肤包
- 全局快捷键切换皮肤（Ctrl+Shift+S）

### 5. HTTP Push API
- `localhost:9527` 接收外部工具状态推送
- Token 校验安全机制
- 支持 Claude Code Hook 接入

### 6. 多窗口架构
- 宠物窗口（常驻桌面）
- 仪表盘窗口（按需打开，查看详细状态）
- IPC 共享状态

## 技术栈

| 层 | 技术 |
|---|---|
| 运行时 | Electron 36 |
| 构建 | electron-vite + Vite 6 + electron-builder |
| UI | React 18 + TypeScript 5 |
| 状态管理 | Zustand 5 |
| 持久化 | electron-store |
| 动画 | Canvas 2D (requestAnimationFrame) |

---

## 创新点

1. **AI 状态可视化** — 将抽象的 AI 工作状态转化为直观的桌面宠物动画，让开发者实时感知 AI 的忙碌程度

2. **插件化架构** — 支持接入任意 AI 工具（Ollama、Claude、GitHub Copilot 等），扩展性强

3. **开放 API** — 提供 HTTP Push 接口，外部工具可轻松接入状态推送

4. **皮肤生态** — 支持自定义皮肤包，用户可根据喜好更换宠物形象

- 本地运行 Ollama 等 AI 模型的用户
- 需要监控 GitHub 活动的开发者
